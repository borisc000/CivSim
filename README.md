# CivSim

Este repo ahora tiene dos rutas:

1. MVP web inicial en `index.html` (`game.js`, `style.css`)
2. MVP principal en Godot 4 dentro de `godot/`

## Recomendacion actual

Usar la version Godot como base de desarrollo para escalar el proyecto.

- Guia de arranque: `godot/README.md`
- Proyecto Godot: `godot/project.godot`
